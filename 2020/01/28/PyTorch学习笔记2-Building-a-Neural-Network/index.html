Basic steps for building a Neural Network

Concluded from the VAE code.

  1. process the data

    1. decide what dataset to use
    2. download the data and transform them to tensor
    3. define BATCH_SIZE to make an iterator from the tensor-form data

  2. build the NN architecture

    1. def __init__( )：decide what hyper-params to input (i.e. the input_dim , output_dim, hidden layers and their dim. It’s used when you create an instance)
    2. def __init__( )：define the weight matrix for between each pair of layers (eg. Linear(dim1, dim2) )
    3. def __forward__( )：define the forward prop (i.e. decide which activate function to use w.r.t. each layer. Sometimes this step may be complexer, eg. including sampling at some layers, like VAE)

  3. define the specific hyper-params to create an instance, as well as the optimizer

  4. define the train() function

    1. instance.train()

    2. initiate the train_loss

    3. Loop: iterate(enumerate) the data

      1. reshape the data

      2. zero the grad

      3. feed the data to the model instance and forward prop to get the output

      4. define the loss between the output and target output

      5. accumulate the train_loss, and backprop

      6. update the params (i.e. optimizer.step() )

  5. define the test() function

  6. Loop for train_epoch to train and eval the model instance

Hyper_params of high frequency:

  • learning rate (used by optimizer)
  • dimensions of each layer
  • batch_size
  • train_epoch (numbers of training times)

When you use negative sampling, your loss might be compute in NN.forward( ) ！！！！！

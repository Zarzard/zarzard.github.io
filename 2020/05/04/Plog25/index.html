Zarzard’s Plog25

Mon

人生第一节网课是院长的泛函分析。我没怎么预习，加上老师赶进度讲得飞快，我听得快睡着了。过几天数据库和微分几何开了就更忙了，得抓紧预习，抓紧整理笔记。

当下第一任务是课内，适应网课节奏之后，有时间再来看VI和强化学习。

而且还有推荐信和英文陈述要写。

目前几大任务：

  1. 适应并跟上网课节奏，整理笔记，完成作业
  2. 本周内搞定推荐信+英文陈述
  3. 每天晚上日总结+脱水+阅读
  4. 复习概率论与数理统计，实变，多元微积分

Wed

早上6点半起来快速预习了泛函，最后也只预习了第一节课的内容，第二节课开始就开始有点跟不上了。看来下次要预习得更快一些，听课注意力要更集中才行！！

这三天除了泛函还是泛函，有点难顶。而且目前光是跟着推一遍证明就已经 se i bai 了，接下来还要思考直观意义和为什么要这么做。然后开始做作业。

啊啊啊明天早上还是继续把泛函搞定，明天下午开始查 VI 方向论文。明天晚上搞推荐信。

最近脱水玩得有点多，主要是总是浏览别人的想法，还有他们的历史消息（主要是确认性别）。操，明明没必要这样做的，就算知道了那个人是女的，多看几条就会发现她的素质不高，有的矫情，有的毫无条理胡乱发泄，有的缺爱，……然后就很失望。其实根本没必要这样做的，都是荷尔蒙在作怪。我玩脱水的目的就是练习写作，学会表达东西，把没有在朋友圈说出来的东西一吐为快。写得好，自然会有人 follow。不要再浪费时间了！！

Fri

昨天才结束泛函笔记整理，今天马上又来了网课，还是两门：数据库+微分几何。还好微分几何第一节全程吹水，没什么压力。不过最近数学又多了一个任务：阅读《流形上的微积分》前两章。

事情多起来了，先列一列：

  • 泛函作业（其他题也看一看）
  • 李宏毅强化学习（大概需要一整天）+RL&VI结合论文
  • 课内复习：概率论数理统计+实变函数+数据结构
  • 推荐信及英文版个人陈述
  • 《流形上的微积分》前两章

本周末任务：泛函作业及预习（2d）；李宏毅强化学习（1d）；推荐信（1d）；英文版个人陈述（1d）；数据库学习（1d）

具体安排：

时间 任务
周六上午 泛函作业
周六下午 数据库
周六晚上 推荐信
周日上午 泛函预习
周日下午 李宏毅强化学习
周日晚上 英文版个人陈述

每天晚上记得总结。脱水没时间就少说点，放几首歌也行。每晚22：50前睡觉，早上6点起。
